# easy_BFopensslCTF

Bash script that given a password (or a wordlist) tries to decrypt an OpenSSL encrypted file using several algorithms.

## Help message
bf_openssl.bash -f input_file [-v] [-a] [-b] [-h] [-t password] [-p pass_file]

  v: Verbose level

  a: All cyphers

  b: Base64
  
  h: Help
  
  t: Use pass
  
  p: Use pass file
